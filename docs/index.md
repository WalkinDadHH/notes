## 推荐阅读顺序

<div class="reading-path" markdown>

1. 第一次进入本站，先看 [数学总览](math/index.md) 和 [专业课程总览](major/index.md)。
2. 如果想直接看完成度最高的内容，先读 [常微分方程](math/ode/index.md)。
3. 如果更关心专业课，先进入 [专业基础课](major/foundation/index.md)，再看工程力学与工程热力学。
4. 如果只是临时复习物理，直接进入 [光学](physics/optics/index.md)。

</div>
