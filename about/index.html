    关于本站¶

    技术栈¶

    组件 说明
    MkDocs 静态网站生成器
    Material for MkDocs 主题
    霞鹜文楷 中文正文字体
    JetBrains Mono 代码字体

    构建与部署¶

    本地预览¶

    # 安装依赖
pip install mkdocs-material

# 启动开发服务器
mkdocs serve

# 浏览器打开 http://127.0.0.1:8000

    构建静态文件¶

    mkdocs build
# 输出到 site/ 目录

    部署到 GitHub Pages¶

    mkdocs gh-deploy
    Made with Material for MkDocs
